Alpaca

Accept Bitcoin on your website or app. Made for developers.

Who is this for?

Alpaca is made for developers. It ships a server and client SDK to easily integrate Bitcoin payments into a website or app. Alpaca intends to be like Stripe but for Bitcoin.

Features

  • Free and open source software.
  • Get paid directly into your wallet. No third parties. Non-custodial. No third parties touch your coins.
  • Easy-to-use SDK to integrate Alpaca payments from several programming languages
  • No need to run a fullnode—communicates with Electrum servers by default
  • Embedded SPV implementation verifies transactions trustlessly
  • Fresh addresses for every transaction
  • BIP-32 address generation; implements heirarchical deterministic key derivation
  • Implemented as a lightweight C++ server
  • Cloud-native—suitable for cheap cloud deployments (<$5/month total cost of ownership)
